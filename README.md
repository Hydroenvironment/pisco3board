# Hydroclimate Scientific Dashboard

## Description
Advanced Streamlit-based hydroclimate analysis system for gridded NetCDF precipitation data.

Dataset compatible structure:
- precipitation(Z1, latitude, longitude)

## Features
- Point extraction (lat/lon)
- Polygon analysis (draw or shapefile)
- Climatology
- Anomalies
- Extreme indices (ETCCDI-like)
- SPI-like index
- Event detection
- Interactive maps
- Time series + statistics

## Deployment (Streamlit Cloud)

1. Upload this folder to GitHub
2. Go to https://share.streamlit.io
3. Connect repository
4. Select `app.py`
5. Add NetCDF URL (public Google Drive link)

## Google Drive format
Use direct download link:
https://drive.google.com/uc?id=FILE_ID
